# DecentHolograms-CI

This repository acts as a basic form of CI system, building any changes done on the [DecentHolograms repository](https://github.com/DecentSoftware-eu/DecentHolograms) repository.  
The entire system works solely through GitHub Actions, using releases as "builds" and a `.last_upstream_sha` file to keep track of the previous Commit used to build stuff.

This entire thing is meant as a proof-of-concept (or just a silly thing I did to waste time).
